# UDP Screenshot Sender

## Overview
This project is a learning-oriented application that demonstrates how screenshots can be captured and transmitted over a UDP connection.

The main focus of the project is to explore UDP-based communication and basic image transfer concepts.

---

## Purpose
- Practice UDP networking concepts
- Learn how to capture screenshots programmatically
- Experiment with sending binary data over the network

---

## Technologies
- C#
- .NET
- UDP Sockets
- GDI+ (for screen capture)

---

## Functionality
- Captures a screenshot from the local machine
- Sends screenshot data via UDP
- Demonstrates basic client/server UDP communication

---

## How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the sender and receiver components as required
4. Ensure both sides are configured with the correct IP and port
